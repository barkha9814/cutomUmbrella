# CUSTOM UMBRELLA

## Installation

### 1) Clone the repository:
```
git clone https://github.com/barkha9814/cutomUmbrella.git
```
  

### 2) Dependencies installation
inside root folder, run:
```
npm install
```

### 3) Run the project
inside root folder, run:
```
npm start
```

# Tech Stack for front End
- [ReactJS](https://legacy.reactjs.org/)
